CommLite is tui's serial port tool based CSerialPort

# Platform 平台
CommLite has been tested to the following platforms:

   - DOS ( x86_64 )
   - Windows ( x86_64 )
   - Linux ( x86_64, aarch64, mips64el, s390x, ppc64le )
   - macOS ( x86_64 )
   - Raspberry Pi ( armv7l )
   - FreeBSD ( x86_64 )
   - ...

# Quick Start

```
$ git clone https://github.com/itas109/CommLite.git
$ cd CommLite
$ cmake -B bin .
$ cmake --build bin
```
